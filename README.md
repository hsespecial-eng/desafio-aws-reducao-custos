# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

**Data:** 17 de Janeiro de 2026  
**Empresa:** Abstergo Industries  
**Responsável:** Humberto dos Santos

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries. O objetivo do projeto foi elencar 3 serviços AWS com a finalidade de realizar diminuição de custos imediatos e otimização do orçamento de nuvem.

## Descrição do Projeto

### Etapa 1: 
- **Ferramenta:** AWS Trusted Advisor
- **Foco:** Otimização de custos e segurança.
- **Caso de Uso:** Identificar instâncias EC2 com baixo uso e volumes EBS órfãos para interromper cobranças de recursos não utilizados.

### Etapa 2: 
- **Ferramenta:** Amazon S3 Intelligent-Tiering
- **Foco:** Gerenciamento automático de custo de armazenamento.
- **Caso de Uso:** Mover automaticamente dados de logs e vendas para camadas mais baratas após 30 dias de inatividade, sem intervenção manual.

### Etapa 3: 
- **Ferramenta:** AWS Budgets
- **Foco:** Controle e alertas de gastos.
- **Caso de Uso:** Configuração de alertas automáticos via e-mail quando os custos atingirem 80% do teto mensal estabelecido pela empresa.

## Conclusão
A implementação foca na automação e visibilidade. Espera-se uma redução imediata no desperdício de recursos ociosos e maior previsibilidade financeira para a Abstergo Industries.
